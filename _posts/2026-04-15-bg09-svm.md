Kernel methods allow linear algorithms to learn non-linear decision boundaries by implicitly mapping data to high-dimensional feature spaces.

## The Kernel Trick

Instead of explicitly computing a feature map $\phi(\mathbf{x})$, we define a **kernel function**:

$$
k(\mathbf{x}, \mathbf{x}') = \langle \phi(\mathbf{x}), \phi(\mathbf{x}') \rangle
$$

Common kernels:
- **Linear**: $k(\mathbf{x}, \mathbf{x}') = \mathbf{x}^\top \mathbf{x}'$
- **Polynomial**: $k(\mathbf{x}, \mathbf{x}') = (\mathbf{x}^\top \mathbf{x}' + c)^d$
- **RBF (Gaussian)**: $k(\mathbf{x}, \mathbf{x}') = \exp\left(-\frac{\|\mathbf{x}-\mathbf{x}'\|^2}{2\sigma^2}\right)$

The RBF kernel maps to an **infinite-dimensional** feature space, yet the kernel evaluation is $O(d)$.

**Mercer's theorem**: a function $k$ is a valid kernel if and only if the kernel matrix $K_{ij} = k(\mathbf{x}_i, \mathbf{x}_j)$ is positive semi-definite for any set of inputs.

## Support Vector Machine (SVM)

SVM finds the **maximum-margin** hyperplane separating two classes.

**Primal problem** (hard margin):

$$
\min_{\mathbf{w}, b} \frac{1}{2}\|\mathbf{w}\|^2 \quad \text{s.t.} \quad y_i(\mathbf{w}^\top \mathbf{x}_i + b) \geq 1, \; \forall i
$$

The margin width is $\frac{2}{\|\mathbf{w}\|}$; maximizing margin = minimizing $\|\mathbf{w}\|$.

**Dual problem** (via Lagrange multipliers):

$$
\max_{\boldsymbol{\alpha}} \sum_i \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j \mathbf{x}_i^\top \mathbf{x}_j \quad \text{s.t.} \quad \alpha_i \geq 0, \; \sum_i \alpha_i y_i = 0
$$

Only data points with $\alpha_i > 0$ influence the decision boundary — these are the **support vectors**.

## Soft Margin SVM

Real data is rarely linearly separable. The soft margin allows violations with slack variables $\xi_i$:

$$
\min_{\mathbf{w}, b, \boldsymbol{\xi}} \frac{1}{2}\|\mathbf{w}\|^2 + C\sum_i \xi_i \quad \text{s.t.} \quad y_i(\mathbf{w}^\top \mathbf{x}_i + b) \geq 1 - \xi_i, \; \xi_i \geq 0
$$

$C$ controls the **bias-variance tradeoff**: large $C$ = low bias, high variance (tighter fit).

## Kernel SVM

In the dual, data only appears via inner products $\mathbf{x}_i^\top \mathbf{x}_j$. Replacing with $k(\mathbf{x}_i, \mathbf{x}_j)$ gives a **non-linear SVM** without changing the optimization structure.
